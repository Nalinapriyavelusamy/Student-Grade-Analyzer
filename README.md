# Student-Grade-Analyzer

📝 A simple beginner-level Python project that:

- Takes input of 5 subject marks  
- Calculates total, average, and grade  
- Displays a neat student report  

---

🔧 **Tech Used**  
- Python (no libraries needed)

---

📌 **Sample Output**


📄 STUDENT REPORT
Name: Nalinapriya
Total Marks: 449 / 500
Average: 89.8
Grade: A
